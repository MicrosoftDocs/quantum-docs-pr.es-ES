# <a name="lg-function"></a>Función LG

Espacio de nombres: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Paquete: [Microsoft. Quantum. Standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Calcula el logaritmo en base 2 de un número.

```qsharp
function Lg (input : Double) : Double
```


## <a name="input"></a>Entrada

### <a name="input--double"></a>entrada: [doble](xref:microsoft.quantum.lang-ref.double)

Número real $x $.



## <a name="output--double"></a>Salida: [Double](xref:microsoft.quantum.lang-ref.double)

El logaritmo en base 2 $y = \ log_2 (x) $ tal que $x = 2 ^ y $.
