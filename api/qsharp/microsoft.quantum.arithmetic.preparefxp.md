# <a name="preparefxp-operation"></a>Operación PrepareFxP

Espacio de nombres: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Paquete: [Microsoft. Quantum. Numerics](https://nuget.org/packages/Microsoft.Quantum.Numerics)


Inicializa un número de punto fijo Quantum en una constante clásica.

```qsharp
operation PrepareFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit is Adj + Ctl
```


## <a name="input"></a>Entrada

### <a name="constant--double"></a>constante: [Double](xref:microsoft.quantum.lang-ref.double)

Constante en la que se va a inicializar el número de punto fijo Quantum.


### <a name="fp--fixedpoint"></a>FP: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Número de punto fijo (de tipo FixedPoint) que se va a inicializar.



## <a name="output--unit"></a>Salida: [unidad](xref:microsoft.quantum.lang-ref.unit)
