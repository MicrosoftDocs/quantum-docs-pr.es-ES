Espacio de nombres: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Configura [](https://nuget.org/packages/)


Agrega una constante clásica a un número de punto fijo Quantum.

```qsharp
operation AddConstantFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit
```


## <a name="input"></a>Entrada

### <a name="constant--double"></a>constante: [Double](xref:microsoft.quantum.lang-ref.double)

Constante que se va a agregar al número de punto fijo Quantum.


### <a name="fp--fixedpoint"></a>FP: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Número de punto fijo al que se agregará la constante.
