# <a name="q-libraries"></a>Bibliotecas de Q#

La biblioteca estándar de Q# incluye operaciones, funciones y tipos definidos por el usuario que se usan para desarrollar programas cuánticos. También se incluyen tres bibliotecas adicionales basadas en la biblioteca estándar: bibliotecas de química, de valores numéricos y de aprendizaje automático.

Para más información acerca de las bibliotecas de Q#, consulte <xref:microsoft.quantum.libraries>.
