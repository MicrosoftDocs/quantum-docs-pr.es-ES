# <a name="littleendian-user-defined-type"></a>Tipo definido por el usuario LittleEndian

Espacio de nombres: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Paquete: [Microsoft. Quantum. Standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Registra que codifica un entero sin signo en orden Little-Endian. El qubit con el índice `0` codifica el bit más bajo de un entero sin signo.

```qsharp

newtype LittleEndian = (Qubit[]);
```
