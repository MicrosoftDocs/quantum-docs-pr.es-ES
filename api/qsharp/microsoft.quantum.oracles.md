# <a name="microsoftquantumoracles-namespace"></a>Espacio de nombres Microsoft. Quantum. Oracle

Este espacio de nombres proporciona tipos definidos por el usuario para decorar varios tipos de Oracle según su uso previsto en algoritmos Quantum como la estimación de la fase y la amplificación de amplitud.
