Espacio de nombres: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Registra que codifica un entero sin signo en orden Big-Endian. El qubit con el índice `0` codifica el bit más alto de un entero sin signo.
