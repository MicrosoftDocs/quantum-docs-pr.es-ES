Espacio de nombres: [Microsoft. Quantum. aritmético](xref:Microsoft.Quantum.Arithmetic)

Paquete: [Microsoft. Quantum. Numerics](https://nuget.org/packages/Microsoft.Quantum.Numerics)


Mida un número de punto fijo, devuelve su valor como Double y restablece el registro en cero.

```qsharp
operation MeasureFxP (fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Double
```

### <a name="fp--fixedpoint"></a>FP: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)
