Convierte un `Result[]` tipo en un `Bool[]` tipo, donde `One` se asigna a `true` y `Zero` se asigna a `false` .

```qsharp
function ResultArrayAsBoolArray (input : Result[]) : Bool[]
```

### <a name="input--__invalidresult__"></a>entrada: __no <Result> válido__[]

## <a name="output--bool"></a>Salida: [bool](xref:microsoft.quantum.lang-ref.bool)[]

`Bool[]` que representa el `input`.
