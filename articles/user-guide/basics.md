<span data-ttu-id="f50d8-119">Lo que realmente *hacen* estas operaciones solo se convierte en concreto por el equipo de destino que se usa para ejecutar el programa de preguntas y respuestas determinado.</span><span class="sxs-lookup"><span data-stu-id="f50d8-119">What these operations actually *do* is only made concrete by the target machine used to run the particular Q# program.</span></span>
