# <a name="what-is-q"></a><span data-ttu-id="9b7c4-103">¿Qué es Q#?</span><span class="sxs-lookup"><span data-stu-id="9b7c4-103">What is Q#?</span></span>

<span data-ttu-id="9b7c4-106">El lenguaje de programación Q# proporciona un intuitivo conjunto de tipos, operaciones y expresiones lógicas para desarrollar algoritmos sin tener que preocuparse por la lógica interna del equipo cuántico.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-106">The Q# programming language provides you with an intuitive set of types, operations and logic expressions to develop algorithms without having to worry about the internal logic of the quantum computer.</span></span>

## <a name="code-algorithms"></a><span data-ttu-id="9b7c4-107">Algoritmos de código</span><span class="sxs-lookup"><span data-stu-id="9b7c4-107">Code algorithms</span></span>

<span data-ttu-id="9b7c4-108">Al principio, los algoritmos de computación cuántica se visualizaban como diagramas, de forma similar a los diagramas de circuitos de la computación clásica.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-108">In the early days of quantum computing algorithms were visualized as diagrams similarly to circuit diagrams in classical computing.</span></span>  <span data-ttu-id="9b7c4-109">Mientras que el modelo de circuitos se ha usado mucho en la investigación de computación cuántica durante muchos años, en Microsoft creemos que los desarrolladores pueden ir más allá de los circuitos cuánticos y desarrollar aplicaciones y algoritmos cuánticos con Q#.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-109">While the circuit model has been very useful for many years in quantum computing research, here at Microsoft, we believe that developers can go beyond quantum circuits and develop quantum algorithms and applications using Q#.</span></span> <span data-ttu-id="9b7c4-110">El lenguaje Q# se ha diseñado para aprovechar lo que hemos aprendido durante décadas de desarrollo de software clásico y capacita a los desarrolladores cuánticos con funcionalidades de lenguaje de alto nivel destinadas específicamente para la computación cuántica.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-110">The Q# language was built to take advantage of what we’ve learned through decades of classical software development, and empower quantum developers with high-level language functionality specifically targeted for quantum computing.</span></span>


## <a name="how-does-q-work"></a><span data-ttu-id="9b7c4-111">¿Cómo funciona Q#?</span><span class="sxs-lookup"><span data-stu-id="9b7c4-111">How does Q# work?</span></span>

<span data-ttu-id="9b7c4-112">Uno de los bloques de compilación fundamentales de Q# es el tipo `Qubit`, que no se puede copiar ni acceder a él directamente, como un qubit real.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-112">One of the fundamental building blocks of Q# is the `Qubit` type, which cannot be copied or directly accessed, just like a real qubit.</span></span> <span data-ttu-id="9b7c4-113">En su lugar, podemos medirlo y almacenar el resultado de la medición en una variable `Result`, un tipo de Q# que puede tomar dos valores posibles: `Zero` y `One`.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-113">Instead, we can measure it and store the outcome of the measurement in a `Result` variable, a Q# type that can take two possible values: `Zero` and `One`.</span></span> <span data-ttu-id="9b7c4-114">Construcciones como esta garantizan que los algoritmos respeten siempre las leyes de la física cuántica, así como que se puedan ejecutar correctamente en simuladores o equipos cuánticos.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-114">Constructs like this one guarantee that algorithms always respect the laws of quantum physics and can run correctly on quantum computers or simulators.</span></span>

<span data-ttu-id="9b7c4-115">Q# también incluye funcionalidades de lógica clásicas, como condicionales o bucles con algunos matices, para asegurarse de que se respetan todas las reglas cuánticas.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-115">Q# also includes classical logic features like conditionals or loops with some subtleties to make sure that all the quantum rules are being respected.</span></span> <span data-ttu-id="9b7c4-116">Por ejemplo, las operaciones cuánticas deben ser reversibles.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-116">For example, quantum operations need to be reversible.</span></span> <span data-ttu-id="9b7c4-117">Esto aplica algunas restricciones en la forma en que se ejecutan los bucles.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-117">This enforces some constraints on the way loops are executed.</span></span>

<span data-ttu-id="9b7c4-118">Con frecuencia, los programas de Q# están emparejados con un programa host escrito en C# o Python, que puede proporcionar una organización adecuada del código clásico y cuántico.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-118">Q# programs are often paired with a host program written in C# or Python, which can provide convenient organization of classical and quantum code.</span></span> <span data-ttu-id="9b7c4-119">Además de la compatibilidad con lenguajes .NET, como C# y Python, QDK proporciona compatibilidad de Jupyter Notebook con el kernel de Jupyter IQ#.</span><span class="sxs-lookup"><span data-stu-id="9b7c4-119">In addition to supporting .NET languages such as C# and Python, the QDK provides Jupyter Notebook support with the IQ# Jupyter kernel.</span></span>

## <a name="use-q-to-design-quantum-algorithms"></a><span data-ttu-id="9b7c4-123">Uso de Q# para diseñar algoritmos cuánticos</span><span class="sxs-lookup"><span data-stu-id="9b7c4-123">Use Q# to design quantum algorithms</span></span>
