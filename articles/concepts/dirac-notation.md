# <a name="dirac-notation"></a><span data-ttu-id="32db2-182">Notación Dirac</span><span class="sxs-lookup"><span data-stu-id="32db2-182">Dirac Notation</span></span>

<span data-ttu-id="32db2-183">Aunque la notación de vectores de columna es omnipresente en álgebra lineal, a menudo resulta incómodo en la informática Quantum, especialmente cuando se trabaja con varios qubits.</span><span class="sxs-lookup"><span data-stu-id="32db2-183">While column vector notation is ubiquitous in linear algebra, it is often cumbersome in quantum computing especially when dealing with multiple qubits.</span></span>  <span data-ttu-id="32db2-184">Por ejemplo, cuando se define $ \psi $ como un vector, no se desactiva explícitamente si $ \psi $ es una fila o un vector de columna.</span><span class="sxs-lookup"><span data-stu-id="32db2-184">For example, when we define $\psi$ to be a vector it is not explicitly clear whether $\psi$ is a row or a column vector.</span></span>  <span data-ttu-id="32db2-185">Por lo tanto $ \phi $ , si y $ \psi $ son vectores, es igualmente inclaro si $ \phi \psi $ se define incluso porque las formas de $ \phi $ y $ \psi $ pueden no estar claras en el contexto.</span><span class="sxs-lookup"><span data-stu-id="32db2-185">Thus if $\phi$ and $\psi$ are vectors then it is equally unclear if $\phi \psi$ is even defined because the shapes of $\phi$ and $\psi$ may be unclear in the context.</span></span>  <span data-ttu-id="32db2-186">Además de la ambigüedad sobre las formas de vectores, la expresión de vectores simples con la notación algebraico lineal introducida anteriormente puede ser muy engorroso.</span><span class="sxs-lookup"><span data-stu-id="32db2-186">Beyond the ambiguity about the shapes of vectors, expressing even simple vectors using the linear algebraic notation introduced earlier can be very cumbersome.</span></span> <span data-ttu-id="32db2-187">Por ejemplo, si deseamos describir un $ $ Estado de n-qubit en el que cada qubit toma el valor $ 0 $ , se expresaría formalmente el estado como</span><span class="sxs-lookup"><span data-stu-id="32db2-187">For example, if we wish to describe an $n$-qubit state where each qubit takes the value $0$ then we would formally express the state as</span></span> 

<span data-ttu-id="32db2-188">$$\begin{bmatrix}1 \\\\ 0 \end{bmatrix} \otimes \cdots \otimes \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} .</span><span class="sxs-lookup"><span data-stu-id="32db2-188">$$\begin{bmatrix}1 \\\\  0 \end{bmatrix}\otimes \cdots \otimes\begin{bmatrix}1 \\\\  0 \end{bmatrix}.</span></span> $$  

<span data-ttu-id="32db2-189">Por supuesto, la evaluación de este producto tensores es poco práctica porque el vector se encuentra en un espacio exponencialmente grande y, por lo tanto, esta notación es, de hecho, la mejor descripción del estado que se puede proporcionar mediante la notación anterior.</span><span class="sxs-lookup"><span data-stu-id="32db2-189">Of course evaluating this tensor product is impractical because the vector lies in an exponentially large space and so this notation is in fact the best description of the state that can be given using the previous notation.</span></span>  

<span data-ttu-id="32db2-190">La [*notación Dirac*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) resuelve estos problemas presentando un nuevo lenguaje para ajustarse a las necesidades precisas de la mecánica de Quantum.</span><span class="sxs-lookup"><span data-stu-id="32db2-190">[*Dirac notation*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) solves these issues by presenting a new language to fit the precise needs of quantum mechanics.</span></span>  <span data-ttu-id="32db2-191">Por esta razón, se recomienda que el lector no vea los ejemplos de esta sección como una receta rígida de cómo describir los Estados de Quantum, sino que anime al lector a verlos como sugerencias que se pueden usar para expresar ideas de Quantum de manera concisa.</span><span class="sxs-lookup"><span data-stu-id="32db2-191">For this reason, we recommend the reader not view the examples in this section as a rigid prescription of how to describe quantum states, but rather encourage the reader to view these as suggestions that can be used to concisely express quantum ideas.</span></span>

<span data-ttu-id="32db2-192">Hay dos tipos de vectores en notación Dirac: el vector *Bra* y el vector *les* , por lo que el nombre se debe a que, cuando se unen, forman un *freno* o producto interno.</span><span class="sxs-lookup"><span data-stu-id="32db2-192">There are two types of vectors in Dirac notation: the *bra* vector and the *ket* vector, so named because when put together they form a *braket* or inner product.</span></span>  <span data-ttu-id="32db2-193">Si $ \psi $ es un vector de columna, podemos escribirlo en la notación Dirac como $ \ket { \psi } $ , donde el $ \ket { \cdot } $ indica que se trata de un vector de columna unitaria, es decir, un vector *les* .</span><span class="sxs-lookup"><span data-stu-id="32db2-193">If $\psi$ is a column vector then we can write it in Dirac notation as $\ket{\psi}$, where the $\ket{\cdot}$ denotes that it is a unit column vector, i.e., a *ket* vector.</span></span>  <span data-ttu-id="32db2-194">Del mismo modo, el vector de fila $ \psi ^ \dagger $ se expresa como $ \bra { \psi } $ .</span><span class="sxs-lookup"><span data-stu-id="32db2-194">Similarly, the row vector $\psi^\dagger$ is expressed as $\bra{\psi}$.</span></span> <span data-ttu-id="32db2-195">En otras palabras, $ \psi ^ \dagger $ se obtiene aplicando la conjugación compleja de modo de entrada a los elementos de la transposición de $ \psi $ .</span><span class="sxs-lookup"><span data-stu-id="32db2-195">In other words, $\psi^\dagger$ is obtained by applying entry-wise complex conjugation to the elements of the transpose of $\psi$.</span></span> <span data-ttu-id="32db2-196">La notación bra-les implica directamente que $ \braket { \psi | \psi } $ es el producto interno de vector $ \psi $ con sí mismo, que es por definición $ 1 $ .</span><span class="sxs-lookup"><span data-stu-id="32db2-196">The bra-ket notation directly implies that $\braket{\psi|\psi}$ is the inner product of vector $\psi$ with itself, which is by definition $1$.</span></span>  

<span data-ttu-id="32db2-197">En general, si $ \psi $ y $ \phi $ son vectores de estado Quantum, su producto interno es, $ \braket { \phi | \psi } $ lo que implica que la probabilidad de medir el estado $ \ket { \psi } $ $ \ket { \phi } $ es $ | \braket { \phi | \psi } | ^ 2 $ .</span><span class="sxs-lookup"><span data-stu-id="32db2-197">More generally, if $\psi$ and $\phi$ are quantum state vectors their inner product is $\braket{\phi|\psi}$ which implies that the probability of measuring the state $\ket{\psi}$ to be $\ket{\phi}$ is $|\braket{\phi|\psi}|^2$.</span></span>  

<span data-ttu-id="32db2-198">La Convención siguiente se usa para describir los Estados de Quantum que codifican los valores de cero y uno (los Estados de base de cálculo de qubit único):</span><span class="sxs-lookup"><span data-stu-id="32db2-198">The following convention is used to describe the quantum states that encode the values of zero and one (the single-qubit computational basis states):</span></span>

$$
<span data-ttu-id="32db2-199">\begin{bmatrix}1 \\\\ 0 \end{bmatrix} = \ket { 0 } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="32db2-199">\begin{bmatrix} 1 \\\\  0 \end{bmatrix} = \ket{0},\qquad</span></span>
<span data-ttu-id="32db2-200">\begin{bmatrix}0 \\\\ 1 \end{bmatrix} = \ket { 1 } .</span><span class="sxs-lookup"><span data-stu-id="32db2-200">\begin{bmatrix} 0 \\\\  1 \end{bmatrix} = \ket{1}.</span></span>
$$

### <a name="example-representing-the-hadamard-operation-with-dirac-notation"></a><span data-ttu-id="32db2-201">Ejemplo: representar la operación Hadamard con la notación Dirac</span><span class="sxs-lookup"><span data-stu-id="32db2-201">Example: representing the Hadamard operation with Dirac notation</span></span>

<span data-ttu-id="32db2-202">La siguiente notación se usa a menudo para describir los Estados que son el resultado de aplicar la puerta Hadamard a $ \ket { 0 } $ y $ \ket { 1 } $ (que corresponden a los vectores de unidad de las $ direcciones + x $ y $ -x $ en la esfera Bloch):</span><span class="sxs-lookup"><span data-stu-id="32db2-202">The following notation is often used to describe the states that result from applying the Hadamard gate to $\ket{0}$ and $\ket{1}$ (which correspond to the unit vectors in the $+x$ and $-x$ directions on the Bloch sphere):</span></span>

$$
<span data-ttu-id="32db2-203">\frac{1 } { \sqrt { 2 } } \begin{bmatrix} 1 \\\\ 1 \end{bmatrix} = H \ket { 0 } = \ket { + } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="32db2-203">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  1 \end{bmatrix}=H\ket{0} = \ket{+},\qquad</span></span>
<span data-ttu-id="32db2-204">\frac{1 } { \sqrt { 2 } } \begin{bmatrix} 1 \\\\ -1 \end{bmatrix} = H \ket { 1 } = \ket { - } .</span><span class="sxs-lookup"><span data-stu-id="32db2-204">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  -1 \end{bmatrix} =H\ket{1} = \ket{-} .</span></span>
$$

<span data-ttu-id="32db2-205">Estos Estados también se pueden expandir mediante la notación Dirac como sumas de $ \ket { 0 } $ y $ \ket { 1 } $ :</span><span class="sxs-lookup"><span data-stu-id="32db2-205">These states can also be expanded using Dirac notation as sums of $\ket{0}$ and $\ket{1}$:</span></span>

$$
<span data-ttu-id="32db2-206">\ket{+}= \frac{ 1 } { \sqrt { 2 } } ( \ket { 0 }  +  \ket { 1 } ), \qquad \ket { - } = \frac { 1 } { \sqrt { 2 } } ( \ket { 0 }  -  \ket { 1 } ).</span><span class="sxs-lookup"><span data-stu-id="32db2-206">\ket{+} = \frac{1}{\sqrt{2}}(\ket{0} + \ket{1}),\qquad \ket{-} = \frac{1}{\sqrt{2}}(\ket{0} - \ket{1}).</span></span>
$$

### <a name="computational-basis-vectors"></a><span data-ttu-id="32db2-207">Vectores de base de cálculo</span><span class="sxs-lookup"><span data-stu-id="32db2-207">Computational basis vectors</span></span>
<span data-ttu-id="32db2-208">Esto demuestra por qué estos Estados se suelen denominar *base de cálculo*: cada estado de cuanto se puede expresar siempre como suma de los vectores de base de cálculo y estas sumas se expresan fácilmente mediante la notación Dirac.</span><span class="sxs-lookup"><span data-stu-id="32db2-208">This demonstrates why these states are often called a *computational basis*: every quantum state can always be expressed as sums of computational basis vectors and such sums are easily expressed using Dirac notation.</span></span>  <span data-ttu-id="32db2-209">Lo contrario también es cierto en que los Estados $ \ket { + } $ y $ \ket { - } $ también forman una base para los Estados de Quantum.</span><span class="sxs-lookup"><span data-stu-id="32db2-209">The converse is also true in that the states $\ket{+}$ and $\ket{-}$ also form a basis for quantum states.</span></span>  <span data-ttu-id="32db2-210">Puede ver esto en el hecho de que</span><span class="sxs-lookup"><span data-stu-id="32db2-210">You can see this from the fact that</span></span>

$$
